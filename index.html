Intro

这一页只讲准备路径，避免你一开始就扎进细碎配置。

Voice Typing Contest

一个偏实验室风格的本地 benchmark 工具。

它会递归读取音频测试集的子文件夹，驱动多个 app，用统一输入框接收文本，再把时序和结果写进 SQLite。

01

Prepare

装好 BlackHole，确认 target app 已配置好全局热键。

02

Configure

在左侧导航底部打开 Settings，录入 app、音频根目录和 SQLite 路径。

03

Run

回到 Main 开始自动运行，看高亮步骤和失败提示。

Checklist

开跑前四项都过，失败率会低很多。

  • Accessibility granted
  • Virtual device installed
  • Target apps configured
  • Recursive sample folder ready
